GIT

  • git status: Checks what branch we are currently on

  • git status: Checa em qual dos ramos nos estamos

  • git checkout -b branch-name: Creates a new branch and switches to it

  • git checkou -b branch-name: Cria um novo ramo e abre ele

  • git add -A: It`s used to commit all files in the working branch

  • git add -A: É usado para gravar todos os arquivos no atual ramo

  • git commit -m "add notes here": We use the -m flag to associate a message with our commit. Once completed, our Git repository is now up to date with our working directory!

  • git commit -m "adicione notas aqui": Usamos a bandeira -m para associar uma mensagem ao nosso commit. Uma vez concluído, nosso repositório Git agora está atualizado com nosso diretório de trabalho!

  • git pull origin main: It`s used to pull the origin branch, It is a good habit to always pull the base branch into your feature branch before pushing up any code.

  • git pull origin main: É usado para puxar o ramo de origem, É um bom costume de sempre puxar o ramo original antes de enviar qualquer código
